Financial Modeling

We utilized the Black-Scholes model for option pricing evaluations this quarter:

\[ C = S_t N(d_1) - K e^{-rt} N(d_2) \]

Where volatility calculations were derived as:

\[ d_1 = \frac{\ln(\frac{S_t}{K}) + (r + \frac{\sigma^2}{2})t}{\sigma\sqrt{t}} \]
Explain the variables: S is current stock price, K is strike price, r is risk-free rate.

import pandas as pd
import numpy as np

def detect_anomalies(transaction_data):
    # Calculate Z-Score for transaction amounts
    threshold = 3
    z_scores = np.abs((transaction_data - transaction_data.mean()) / transaction_data.std())
    
    return z_scores > threshold
